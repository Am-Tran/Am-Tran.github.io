📊 Portfolio Data Analyst
Bienvenue sur le dépôt de mon portfolio. Ce projet centralise mes travaux d'analyse, d'automatisation et de visualisation de données. L'objectif est de démontrer ma capacité à transformer des données brutes en insights exploitables.

🚀 Projets mis en avant
PathFinder : Pipeline ETL automatisé de monitoring du marché de la Data (Python, Selenium, Streamlit).

CineCreuse : Moteur de recommandation de films basé sur la similarité cosinus (Machine Learning, Scikit-Learn).

Stratégie Vélo : Analyse de performance commerciale et pilotage de lancement (SQL, Power BI).

🛠️ Stack Technique
Langages & Analyse : Python (Pandas, Scikit-Learn), SQL.

Visualisation : Power BI, Streamlit.

Web (Structure) : HTML5, CSS3, JavaScript.

Librairies UI : FontAwesome (icônes), Swiper.js (carrousel), GLightbox (zoom images).
